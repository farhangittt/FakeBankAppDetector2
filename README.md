# Fake Bank App Detector

A cybersecurity-focused project designed to detect potentially malicious or fake banking applications using static and behavioral analysis techniques.

## Features
- Detects suspicious banking apps
- Analyzes app behavior and patterns
- Identifies potential phishing or malicious indicators
- Lightweight and easy to use

## 🛠️ Tech Stack
- Python
- Android Analysis Tools
- Security Libraries

Live Demo - https://fakebankappdetector2.onrender.com/
